## Project installation

Clone git repository

```
git clone https://github.com/near85dy/ytear-backend.git
```

Install libraries

```
npm install
```

You should create .env file and fill rows

```
BETTER_AUTH_SECRET=
BETTER_AUTH_URL=
DATABASE_URL=
```

After that, you can run project

```
npm run start
```

### Required

- npm
- nodejs
- postgres
